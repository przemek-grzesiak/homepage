CSS - Cascading Style Sheets

CSS - odpowiada za prezentację, wpływa na prezentowanie treści HTML na stronie.

Notatki z CSS

Selektory CSS

element - standardowe stylowanie po elemencie, bez wyróżnika np. h1 ostyluje wszystkie nagłówki, section wszystkie sekcje.

.class - stylowanie po klasie elementu zaczyna się od kropki np. .name ostyluje wszystkie elementy o klasie "name".

#id - stylowanie po id elementu zaczyna się od hash'a (#) np. #name ostyluje element o id "name".
