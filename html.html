HTML - odpowiada za treść (a właściwie jej odpowiednie opakowanie. HTML to szkielet strony internetowej

Znaczniki HTML

Budowa znacznika

<znacznik> atrybut="wartość"> treść </znacznik >

Podstawowe znaczniki z lekcji HTML 1

  • < main> </main> - Głowna część dokumentu
  • < section> </section> - Sekcja dokumentu
  • < header> </header> - Nagłówek dokumentu lub sekcji
  • < footer> </footer> - Stopka dokumentu lub sekcji
  • < h1 - h6> - Tekstowe nagłówki tytułowe dokumentu lub sekcji
  • < p> </p> - Paragraf
  • < article> </article> - artykuł -> część która może funkcjonować samodzielnie
